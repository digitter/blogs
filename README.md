# ブログ管理

--
## ルール

- 記事、記事画像用ディレクトリを置く場所

>articleディレクトリ直下のディレクトリ内に置く。

> カテゴリ > 記事、画像ディレクトリ

e.g.)

```
article/
  C/
    images/
    記事.md
```

- 命名規則
>articleディレクトリ直下のディレクトリ名はカテゴリーの名前にする。

```
article/
  C/
  Ruby/
  Rails/
  JavaSript/
  React/
```

---

1. Markdownファイルに纏める。
2. Contentfulで推敲する。
3. デプロイする。
4. 必要ならupdateする。
